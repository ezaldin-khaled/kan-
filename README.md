# KAN Agency

Single-page marketing site built from the Stitch design export (`stitch_professional_design_refiner(1)`).

## Stack

- React 19 + Vite
- Tailwind CSS (design tokens from Stitch `DESIGN.md`)
- GSAP + ScrollTrigger for scroll reveals and hero intro

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```
